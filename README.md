# Resume Website – Course Assignment

This website was created to function as an **online resume** for an assignment in my **Web Programming** course as part of the **MSc Data Science and Artificial Intelligence** at the **University of Liverpool**.

The objective of the assignment was to demonstrate understanding of **core web development fundamentals**, including layout, structure, and styling, without relying on external frameworks or scripting.

---

## Purpose
- Serve as a resume-style personal website
- Demonstrate semantic HTML structure and CSS styling
- Apply best practices in layout, responsiveness, and readability
- Fulfill assessed coursework requirements

---

## Technologies used
- **HTML**
- **CSS**

No JavaScript, frameworks, or backend technologies were used.

---

## Academic context
This repository contains coursework completed for **COMP519** as part of the **MSc in Data Science and Artificial Intelligence**.

The module introduced foundational programming and software development concepts.  
Assignments emphasized structured problem-solving, clarity, and maintainable code.

This project reflects those principles applied in a **web-focused context**.
